## Setting up Dovetail UI

Dovetail makes use of [TIBCO Cloud Integration](https://www.tibco.com/products/cloud-integration) as UI.

Just sign in, or apply for a free trial to start modelling.

Once you have access to TIBCO Cloud Integration go to our [labs](../../labs) page to get more detail information.
